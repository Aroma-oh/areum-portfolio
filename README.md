# [🔗 Aroma-oh-Portfolio](https://aroma-oh-portfolio.com/)
프론트엔드 취업을 위한 포트폴리오 웹 페이지입니다. 

## Features
- 마우스 이벤트, 옵저버 패턴을 적용하였습니다.
- firebase를 이용하여 서버에서 데이터를 받아오도록 하였습니다.
